# Webpack: Manipulando modulos na sua webapp
Curso Alura

## Instalação
./client/npm install
./server/npm install

## Execução
./client/npm run watch
./server/npm start

Acessar o endereço no navegador: localhost:8080

## Distribuição
### Produção
./client/npm run build-prod

### Desenvolvimento
./client/npm run build-dev
